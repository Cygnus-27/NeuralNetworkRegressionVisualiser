
Project Development GuidelinesStudents are required to develop a web-based project based on the topics assigned in either Category 1 (Computer Networks) or Category 2 (Machine Learning).1. Concept DescriptionFirst, clearly and precisely describe the concept of the topic.Your explanation should include:Definition of the concept Explanation of the concept stepwiseExplanation of the concept by solving stepwise using a small data input Purpose of the concept Where it is used in real systems Important parameters or variables involved The explanation should be simple, clear, and technically correct.2. Video SectionCreate a Video Box in the webpage.Embed or link a relevant YouTube video explaining the concept. The video should clearly explain the topic selected. 3. Further Reading / ReferencesProvide references for further learning.This must include:At least one prescribed textbook reference Official technical websites Research papers or educational websites if available Providing the direct link to the textbook section or topic webpage is mandatory.Category 1: Computer Networks ProjectsStudents assigned Computer Network topics (for example: signal strength, flow control, IPv4 filtering, simulations, customized UDP, etc.) must implement the following.Visualization SystemDevelop an interactive visualization tool.Requirements:User InputThe system should accept necessary inputs from the user. Examples:Packet size Transmission rate Window size Network parameters Concept Visualization Based on user inputs, the system should visually demonstrate how the concept works. Step-by-Step Explanation The algorithm or process must be shown step by step. Solution Output The system should compute the result and display the output clearly. Virtual Lab Reference You may refer to Virtual Lab given in our website for ideas on visualization. Additional Webpage RequirementsThe webpage must include the following buttons:LearnHelpDownload Developed by Rum simulation (If applicable)Virtual Lab Reference These should allow the user to interact with the simulation easily.Category 2: Machine Learning ProjectsStudents assigned Machine Learning topics should develop a similar interactive system.Data Input OptionsThe system should allow users to:Upload datasets Enter M × N numerical data Upload image datasets (if applicable) Model ExecutionThe application should:Load the dataset Apply the machine learning algorithm Train the model Produce the output results Visualization RequirementThe system should also show:Visualization of input data Visualization of algorithm working Step-by-step execution of the algorithm For example:Cluster formation (for K-Means) Decision boundary visualization Iteration-wise results Step-by-Step Algorithm DemonstrationIt is very important to show how the algorithm works internally.Example:For K-Means Algorithm, show:Initial centroid selection Distance calculation Cluster assignment Centroid update Iteration process Final clusters You can refer to the example files named “K-Means” provided. Those files demonstrate how the algorithm execution should be displayed step-by-step.If you are unable to implement the exact format, try to follow a similar structured explanation. Final ExpectationYour project should include:Concept explanation Video explanation References Interactive visualization User inputs Step-by-step working of the concept/algorithm Final output results
