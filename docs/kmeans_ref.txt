
What Is Clustering? Clustering is a process of partitioning a set of data (or objects) into a set of meaningful sub-classes, called clusters. Help users understand the natural grouping or structure in a data set. Clustering: unsupervised classification: no predefined classes. What is K-Means?It is an unsupervised machine learning algorithm used for clustering.It groups similar data points together. The number of clusters K must be defined before running the algorithm. Each cluster is represented by a centroid (mean point of the cluster). Goal:Minimize the distance between data points and their cluster centroid.2. Objective Function of K-MeansThe algorithm tries to minimize the Sum of Squared Errors (SSE).J=i=1kx∈Ci∣∣x-μi∣∣2Where:k= number of clusters Ci= set of points in cluster iμi= centroid of cluster i∣∣x-μi∣∣2= squared distance between data point and centroid The algorithm iteratively minimizes this function.3. K-Means Algorithm StepsStep 1 — Choose K i.e. Select the number of clusters.Step 2 — Initialize Centroids i.e Randomly choose K data points as the initial centroids.Step 3 — Assign Data Points to Nearest Centroid: Calculate the distance between each point and centroid.Usually Euclidean distance is used d=x2-x1)2+(y2-y1)2Each point joins the cluster with minimum distance.Step 4 — Recalculate Centroids. Compute the mean of all points in the cluster.μ=1ni=1nxiThis becomes the new centroid.Step 5 — RepeatRepeat Step 3 and Step 4 until:Centroids stop changing, or Maximum iterations reached. 4.
